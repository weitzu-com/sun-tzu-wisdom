# Application Playbooks

Use these templates when the user wants practical help. Adapt tone to the user's context and avoid sounding militarized.

## Workplace conflict

Diagnosis:
- `道`: What shared work goal can both sides still accept?
- `地`: What hierarchy, process, or history shapes the conflict?
- `虚实`: Where is resistance strongest, and where is there a low-friction opening?

Strategy:
- Reframe from "who is wrong" to "what condition makes cooperation possible."
- Start with facts and shared purpose.
- Make one specific ask.
- Preserve face where possible; set boundaries where needed.

Words to offer:
```text
我想先把目标对齐：我们都希望这件事按时、低风险地完成。现在卡住的点似乎是 X。我的建议是先用 Y 方式试一轮，明天再看结果。你看还有什么约束我没有看到？
```

## Project delay

Diagnosis:
- Use `作战篇`: prolonged conflict drains resources.
- Use `形篇`: stabilize facts before explaining.
- Use `法`: clarify ownership, dates, dependencies, decision rights.

Strategy:
- Report early, not late.
- Separate cause, impact, options, and ask.
- Offer a recovery plan with tradeoffs.

Words to offer:
```text
目前项目预计会比原计划晚 X 天，主要原因是 A 和 B。影响是 C。现在有两个选择：保范围延期，或砍掉 D 保时间。我建议选择方案 2，因为它能保护核心目标。需要你确认的是取舍方向。
```

## Negotiation

Diagnosis:
- Use `谋攻篇`: do not aim to crush; aim for a durable agreement.
- Use `知彼知己`: map interests and fallback options.
- Use `避实击虚`: address constraints, not ego.

Strategy:
- Define your walk-away point privately.
- Ask diagnostic questions before proposing.
- Trade across interests instead of arguing one number.
- Make concessions conditional and explicit.

Words to offer:
```text
我理解你最看重的是 X。我这边必须保护的是 Y。如果我们把 Z 调整一下，是否能同时满足这两个条件？
```

## Leadership decision

Diagnosis:
- Use `计篇`: compare purpose, timing, terrain, people, process.
- Use `势篇`: design conditions so good execution is natural.
- Use `九变篇`: avoid one-size-fits-all policy.

Strategy:
- State the principle.
- Identify affected stakeholders.
- Choose a reversible pilot when uncertainty is high.
- Assign owner, metric, and review date.

Decision frame:
```text
目标是什么？
最大不可逆风险是什么？
哪项约束最真实？
能否先做一个小规模试验？
谁负责观察结果并调整？
```

## Intimate relationship communication

Diagnosis:
- Use `全胜`: preserving trust matters more than winning the argument.
- Use `道`: return to shared care and safety.
- Use `不因怒而动`: do not send messages from the peak of emotion.

Strategy:
- Start with feeling and need, not accusation.
- Make a small request.
- Invite the other person's reality.
- Pause if the conversation becomes unsafe or contemptuous.

Words to offer:
```text
我不是想争输赢。我在意的是我们遇到分歧时还能彼此尊重。刚才发生 X 的时候，我感到 Y，因为我需要 Z。你愿意和我一起想一个下次更好的处理方式吗？
```

## Personal dilemma

Diagnosis:
- Use `知己`: values, energy, constraints, fear, real desire.
- Use `知彼`: external reality, market, family, timing, opportunity cost.
- Use `先胜`: prepare options before making irreversible change.

Strategy:
- Define what must be protected.
- Create three options: stay and improve, small experiment, transition.
- Choose the next reversible move.

Reflection:
```text
我真正想保护的是什么？
我害怕失去什么？
哪一步可以让我获得更多信息，而不是立刻赌上一切？
三个月后我希望自己感谢今天做了哪件小事？
```

## Promotion writing

Use this when the user asks to promote Sunzi as peace wisdom.

Structure:
- Start with the misconception: Sunzi is often read as victory over others.
- Reframe: the deeper wisdom is reducing the need for destructive conflict.
- Anchor with short phrases: `不战而屈人之兵`, `知彼知己`, `主不可以怒而兴师`.
- Give modern examples: workplace conflict, negotiation, family communication, personal decisions.
- End with a peaceful action: examine, align, preserve, then act.

Tone:
- Clear, warm, and practical.
- Avoid glorifying war.
- Avoid claiming Sunzi solves every problem mechanically; say it offers lenses for judgment.
